¿QUÉ CAMBIÓ?
════════════

ANTES: Tenías que definir categorías fijas en el código
   - Maquillaje
   - Cuidado
   - Fragancias
   - Cuerpo

AHORA: Las categorías se detectan automáticamente
   ✓ Creas una carpeta → Se convierte automáticamente en categoría
   ✓ El nombre de la carpeta es el nombre de la categoría
   ✓ Sin editar código, sin configuración


¿CÓMO FUNCIONA?
═══════════════

El script busca TODAS las carpetas dentro de imagenes/

Ejemplo de estructura:

imagenes/
├── maquillaje/
│   ├── base.jpg
│   ├── labial.jpg
│   └── sombras.jpg
├── cuidado/
│   ├── crema.jpg
│   └── serum.jpg
├── perfumes/          ← Nueva categoría sin hacer nada
│   └── perfume1.jpg
└── accesorios/        ← Otra nueva categoría
    ├── pulsera.jpg
    └── anillo.jpg

Resultado:
   ✓ Se crean 4 botones de categoría automáticamente
   ✓ Cada uno con los productos de su carpeta
   ✓ Sin tocar el código

ORDEN DE LAS CATEGORÍAS
═══════════════════════

Las categorías aparecen en ORDEN ALFABÉTICO (por nombre de carpeta).

Ejemplo - orden de aparición:
   imagenes/accesorios/      → Aparece 1°
   imagenes/cuidado/         → Aparece 2°
   imagenes/fragancias/      → Aparece 3°
   imagenes/maquillaje/      → Aparece 4°
   imagenes/ofertas/         → Aparece 5°

Si quieres cambiar el orden, renombra las carpetas con prefijos:

   imagenes/01-maquillaje/   → Aparece 1°
   imagenes/02-cuidado/      → Aparece 2°
   imagenes/03-fragancias/   → Aparece 3°
   imagenes/04-cuerpo/       → Aparece 4°


SOLUCIONAR PROBLEMAS
════════════════════

P: Ejecuté el script pero no ve mis carpetas
R: 
   • Asegúrate que las carpetas estén en imagenes/
   • No deben estar dentro de otras carpetas
   • Que NO sean carpetas vacías (sin imágenes)

P: El nombre de la categoría está mal formateado
R:
   • Nómbrala sin caracteres especiales
   • Usa guiones o guiones bajos: mejor-nombre-aqui
   • El script lo convertirá automáticamente

P: Agregué una carpeta pero no aparece
R:
   • ¿Tiene imágenes adentro?
   • Si no, el script la ignora (no crea categoría vacía)
   • Agrega al menos una imagen
   • Ejecuta de nuevo: node generate-catalog.js

P: Quiero que una categoría aparezca primero
R:
   • Renómbrala con un prefijo número: 00-maquillaje
   • Ejecuta: node generate-catalog.js
   • Aparecerá primero (00 viene antes que 01)

P: ¿Puedo tener subcarpetas?
R:
   • No, el sistema solo busca una profundidad
   • imagenes/maquillaje/ ✓
   • imagenes/maquillaje/ojos/ ✗ (no funciona)
   • Si necesitas organizar más, crea categorías separadas
